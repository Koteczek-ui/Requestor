# Requestor
A lightweight HTTP request testing tool. Send GET, POST, PUT, PATCH, DELETE, HEAD, and OPTIONS requests with custom headers, body content, and templates.

## Depedencies
- Visual Studio 2022/.NET Desktop Runtime

## Additinal info
- Always latest on https://requestor.code-master.pl
- Build explanation: yyyymmddbb (y=year, m=month, d=day, b=build)
